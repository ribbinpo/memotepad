MVP feature list
Start very small:
1. Small floating note window
2. Always on top
3. Global shortcut to show/hide
4. Auto-save while typing
5. Markdown/plain text support
6. Menu bar icon
7. Local storage

Later add:
- Multiple notes
- Search notes
- Pin/unpin note
- Opacity control
- Resize window
- Sync with iCloud folder or Git
- Export to Markdown
